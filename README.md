# note-aplication

## Project setup
```
npm install
```

## Set your firebase configuration
```
1 - go to /src/api/
2 - rename the file settings.js.example to settings.js
3 - set your firebase configuration
```

### Compiles and hot-reloads for development
```
npm run dev
```
